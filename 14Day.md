### **오늘의 배운 내용**
1. `Destructuring` : 구조를 분해
   <br>*ex)```const [a, b, c] = array, 나머지 패턴: [a, ...rest]...```*
2. `Function(func)` : 함수 (함수 선언문, 함수 표현식)
   <br>*ex)```function name(params){...}, function(params){...}...```*
3. `Spread` : 세 개의 점이 앞에 붙으면 배열이나 객체를 펼쳐서 복사하거나 합침
   <br>*ex)```배열 Spread: [...array], 객체 Spread: {...object}...```*
4. `This` : 현재 문맥에서 자신을 가리키는 객체
   <br>*ex)```메소드: obj.method() /일반함수: function(){this} / 화살표함수: () -> {this}...```* 
5. `Map` : 배열의 각 요소를 변환해서 새로운 배열을 만드는 함수<br>[map(지도)처럼 배열의 각 요소를 "변환/매핑"한다]
   <br>*ex)```set, get, has, delete, clear / 모든 타입의 키 사용 가능```*
6. `Set` : 특별한 종류의 객체를 말함 (중복되지 않는 유일한 값들의 집합에 사용
   <br>*ex)```add, has, delete, clear / 집합연산: 합집합, 교집합, 차집합```*
7. `JSON` : 자바스크립트 객체 문법을 기반으로 한 데이터 교환 방식
<br>[깊은 복사를 많이 사용 / 텍스트 형태로 데이터 저장이나 서버와 주고받을 때 사용]
<br>*ex)```parse: 문자열->객체 / stringify: 객체->문자열 / 함수·undefined는 제외```*
